About the Project

The New Pope is an independent data science initiative analyzing the papal conclave using social network analysis, R programming, and public church data.

This site was created by richirikken using open tools like Gephi, GEXF, and GitHub Pages. All code and data are publicly available.

Scientific Use File

A complete scientific use file with full coding, detailed documentation, and data will be made available on Zenodo soon for reproducibility and further research.
